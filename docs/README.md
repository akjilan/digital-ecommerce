## Adding a new ADR

Copy the template below into `docs/adr/NNN-short-title.md` and fill it in.

```markdown
# ADR-NNN: Title

**Date:** YYYY-MM-DD  
**Status:** Proposed | Accepted | Deprecated | Superseded

## Context

Why does this decision need to be made?

## Decision

What was decided?

## Consequences

What are the trade-offs?
```
